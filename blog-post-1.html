Optimizing Flight Search with FreeFly

Posted on October 1, 2023

In developing FreeFly, I explored ways to optimize flight search functionalities. By integrating the Amadeus Flight API and Seats.aero API, and implementing caching mechanisms, I reduced response times by over 50%.

One of the key challenges was handling the large volume of data and ensuring real-time updates. I utilized asynchronous processing and implemented a caching layer using Redis to store frequently accessed data. This significantly improved the application's performance and user experience.

Another important feature was the real-time alert system. By integrating RabbitMQ and Twilio, users could receive instant notifications via SMS and email when desired flight availability changed. This feature greatly enhanced user engagement and satisfaction.

Deploying the application on Google Cloud Platform (GCP) with services like Google App Engine and Cloud SQL ensured scalability and high availability. This project not only honed my technical skills but also deepened my understanding of cloud infrastructure and microservices architecture.
